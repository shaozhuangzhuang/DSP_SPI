一、LED硬件配置
--------------------------------------------------------------------------------
LED0 -> GPIO0
LED1 -> GPIO1  
LED2 -> GPIO2
LED3 -> GPIO3
LED4 -> GPIO4
LED5 -> GPIO5

二、基本使用方法
--------------------------------------------------------------------------------

1. 点亮/熄灭单个LED：
   
   #include "../Drivers/drv_led.h"
   
   Drv_LED_On(LED0);      // 点亮LED0
   Drv_LED_Off(LED1);     // 熄灭LED1
   Drv_LED_Toggle(LED2);  // 翻转LED2


2. 在任务中使用LED：

   void App_Task_MyTask(void)
   {
       static Uint16 count = 0;
       
       // 二进制计数显示
       Drv_LED_On(LED0);
       Drv_LED_On(LED1);
       Drv_LED_Off(LED2);
       
       count++;
   }


3. 注册LED任务到调度器：

   在app_init.c中添加：
   
   #include "../Framework/task_scheduler.h"
   
   void App_Init(void)
   {
       // ... 其他初始化代码 ...
       
       // 注册LED任务（如果需要更多LED任务）
       Scheduler_RegisterLED1Task(My_LED_Task);
       
       // ... 其他代码 ...
   }


三、LED演示示例（app_led_demo.c）
--------------------------------------------------------------------------------

已提供5种LED效果：

1. LED_Demo_RunningLight()     - 流水灯效果
2. LED_Demo_AllBlink()          - 全部闪烁
3. LED_Demo_AlternateBlink()    - 奇偶交替
4. LED_Demo_BreathingLight()    - 呼吸灯
5. LED_Demo_BinaryCounter(n)    - 二进制计数显示


调用示例（在app_task.c中）：

void App_Task_DataProcess(void)
{
    static Uint16 counter = 0;
    
    // 每20ms调用一次，显示二进制计数
    LED_Demo_BinaryCounter(counter);
    counter++;
    if(counter > 63)  // 6位二进制最大值
    {
        counter = 0;
    }
}


四、默认任务配置
--------------------------------------------------------------------------------

当前默认配置：
- LED1任务（1秒周期）：控制LED0翻转
- LED2任务（3秒周期）：控制LED1翻转
- LED2-5：可自由使用


五、扩展更多LED任务
--------------------------------------------------------------------------------

如需要添加更多周期性LED任务：

1. 在Framework/task_scheduler.h中添加：
   
   void Scheduler_RegisterLED3Task(TaskFunc_t task);


2. 在Framework/task_scheduler.c中实现：
   
   static TaskFunc_t led3_task = 0;
   static Uint32 led3_task_counter = 0;
   
   void Scheduler_RegisterLED3Task(TaskFunc_t task)
   {
       led3_task = task;
   }


3. 在Scheduler_Run()中检查执行


六、常见问题
--------------------------------------------------------------------------------

Q: LED不亮？
A: 检查GPIO配置是否正确（drv_system.c中的GPIO0-5配置）

Q: 如何更改LED闪烁频率？
A: 修改Framework/system_config.h中的周期定义

Q: 可以PWM调光吗？
A: 需要配置PWM模块，当前仅支持开关控制
